## 📌 Project Overview
[cite_start]**OMNI EYE** ek intelligent software platform hai jo **Artificial Intelligence** aur **Face Recognition** technology ka upyog karke missing persons ko detect karne mein madad karta hai[cite: 13]. [cite_start]Is system mein users missing person ki photo aur details upload kar sakte hain, jise AI analyze karta hai[cite: 14, 15].

### ⚠️ The Problem Statement
* [cite_start]Har saal hazaron log missing ho jate hain aur unhe dhundna mushkil hota hai[cite: 18].
* [cite_start]Missing person data centralized nahi hota[cite: 20].
* [cite_start]CCTV footage manually check karne mein bahut samay lagta hai aur investigation slow ho jati hai[cite: 21, 22].

### ✅ Our Proposed Solution
Hamara system ek centralized AI platform hai jo:
1. [cite_start]Missing person ka data aur images collect karta hai[cite: 27].
2. [cite_start]**Face recognition (OpenCV + AI)** ka use karke photos ko encode karta hai[cite: 28, 39].
3. [cite_start]Camera frames aur suspicious images ko analyze karke matches dhundta hai[cite: 29].
4. [cite_start]Match milne par (**>80% confidence**) family aur police ko automatic alert bhejta hai[cite: 30, 50].

## 🔄 System Workflow
1. [cite_start]**Registration:** User missing person ki photo aur details submit karta hai[cite: 42].
2. [cite_start]**AI Encoding:** System uploaded face ki unique encoding generate karta hai[cite: 43].
3. [cite_start]**Face Comparison:** AI system suspicious images aur CCTV frames ke sath faces compare karta hai[cite: 45].
4. [cite_start]**Smart Alerting:** Match milne par SMS, Email aur Police Dashboard par notification jati hai[cite: 51, 52, 53, 54].
